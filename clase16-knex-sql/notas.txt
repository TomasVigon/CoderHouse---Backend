SQLite es una biblioteca en lenguaje C  que implementan un motor de base de datos SQL pequeño, rapido,
de alta confiabilidad y con todas las funciones. Su codigo fuente es de dominio publico
El formato de archivo SQLite es estable, multiplataforma y compatible con versiones anteriores

SQLite es como una base de datos pequeña. Es un archivo que almacena la informacion en binario y maneja 
las operaciones CRUD. Tiene todas las ventajas de una BD pero local.
